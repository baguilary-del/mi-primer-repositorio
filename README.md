# Mi primer repositorio 

Este repositorio contiene un conjunto de datos de ejemplo diseñado para prácticas de análisis, visualización y manejo de información comercial. Los archivos incluyen información básica de clientes y registros de ventas asociados.

---

## clientes.csv — Resumen

El archivo **clientes.csv** contiene información general de los clientes registrados.  
Incluye datos como:

- Identificador único del cliente  
- Nombre y apellido  
- Correo electrónico  
- Número de teléfono  
- Ciudad de residencia  

Este conjunto de datos permite identificar características básicas de cada cliente y relacionarlos posteriormente con sus compras.

---

## ventas.csv — Resumen

El archivo **ventas.csv** registra las transacciones realizadas.  
Incluye:

- Identificador de la venta  
- Referencia al cliente que realizó la compra  
- Fecha de la transacción  
- Producto adquirido  
- Cantidad y precio unitario  
- Total de la venta  

Este archivo sirve para analizar el comportamiento de compras, productos más vendidos y valores totales generados.
